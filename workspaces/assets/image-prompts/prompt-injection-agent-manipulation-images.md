# Visual Strategy: Prompt Injection (The Trojan Horse)

This guide shows how to create a "Hidden Command" visual representing indirect prompt injection and data hijacking using high-end glowing code and glass mechanics.

---

### 1. Visual Intent: "Poison in the Stream"
**Concept**: A professional studio photograph of "The Trojan Code." A sleek, massive stream of perfectly organized, bright white and cyan binary data (The User Prompt) is flowing through a dark, transparent glass pipe. However, perfectly camouflaged and hidden directly within the middle of the clean white data stream is a jagged, aggressive, glowing blood-red sequence of characters (The Prompt Injection). The glass pipe is beginning to crack slightly right where the red code is touching it, indicating that the system's structural integrity is failing from the inside.

**Target Vibe**: 
- Stealthy and dangerous
- Internal corruption
- High-end cybersecurity aesthetic
- Contrast between "Clean White Data" and "Corrupt Red Command"

---

### 2. Direct Gemini Prompt
> **Prompt**: "Professional cybersecurity conceptual photography in a dark studio. A thick, perfectly transparent glass pipe is carrying a massive, organized stream of glowing bright white data code and light particles from left to right. Perfectly hidden and embedded exactly inside the middle of the clean white data stream is a jagged, aggressive sequence of glowing blood-red code, representing a malicious prompt injection. Right where the red code touches the inside of the glass pipe, micro-fractures and cracks are beginning to spiderweb outward through the glass, showing the system's failure from within. Cinematic lighting, deep shadows, premium tech aesthetic. Sharp focus on the glowing red code and the cracking glass. No text, no artifacts. Shot on 100mm macro lens, f/2.8. 8k resolution, photorealistic glass, light streams, and fracture textures."

---

### 3. Step-by-Step Gemini Interaction Guide

**Step 1: The Corrupt Stream**
Submit the initial prompt. You are looking for a sense of "Invisible Sabotage."

**Step 2: Refining the "Hidden Nature"**
If the red code looks like a giant, separate block smashing the pipe, guide Gemini:
> *“The red code must look stealthy. It shouldn't look like a separate projectile. It must look like it is originally part of the white data stream itself—a wolf in sheep's clothing. It is flowing smoothly inside the pipe, but its malicious color and jagged shape distinguish it, and it is causing the glass to shatter from the inside out.”*

**Step 3: Controlling the "Glass Damage"**
If the glass pipe is completely exploded, ask for subtlety:
> *“Tone down the destruction. The glass shouldn't be exploding outward. We just want to see delicate, dangerous spiderweb cracks radiating from the red code. This represents a subtle hack bypassing the structural guardrails of the AI, not a bomb going off.”*

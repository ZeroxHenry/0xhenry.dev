# 네이버 블로그 발행 가이드 (수동)

## 왜 수동인가?
네이버는 봇 감지가 강력함. 자동 입력/발행 시 계정 정지 위험.
Chrome 자동화는 읽기 전용(페이지 확인, 셀렉터 탐색)으로만 사용.

---

## 발행 순서

### 1. 포스트 파일 열기
`generated/2026-04/` 에서 발행할 `.md` 파일을 열어 `===COPY===` 블록을 찾음.

### 2. 네이버 블로그 글쓰기
https://blog.naver.com/0xhenry/postwrite 접속

### 3. 제목 복붙
파일의 `[제목]` 블록 복사 → 에디터 제목 필드에 붙여넣기

### 4. 본문 복붙
파일의 `[본문]` 블록을 **섹션 단위로** 복사 → 에디터에 붙여넣기
- 한 번에 전체 붙여넣기 OK (수동이니 봇 감지 걱정 없음)
- H2 소제목은 에디터에서 "제목2" 서식 적용

### 5. 이미지 삽입
`[이미지 N]` 위치에 해당 이미지 파일 삽입
- 에디터 상단 "사진" 버튼 클릭
- `generated/images/YYYY-MM/` 에서 파일 선택

### 6. 카테고리 선택
파일에 명시된 카테고리 선택

### 7. 태그 입력
파일의 `[태그]` 블록 복사 → 태그 입력란에 하나씩 입력
(# 빼고 단어만 입력, Enter로 구분)

### 8. 발행 설정
- 공개: 전체공개
- 검색허용: ON
- CCL: 기본값

### 9. 발행 버튼 클릭

### 10. 발행 후
- 발행된 URL 복사
- Claude에게 알려주면 index.json 업데이트

---

## 포스트 파일 구조

각 포스트 .md 파일은 아래 형태로 되어 있음:

```
(frontmatter — 메타 정보)

===COPY: 제목===
제목 텍스트
===END===

===COPY: 본문===
본문 전체 (소제목, 문단 포함)
===END===

===COPY: 태그===
태그1, 태그2, 태그3, ...
===END===

(이미지 명세 — Gemini 프롬프트)
```

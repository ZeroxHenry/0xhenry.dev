# 네이버 AI 블로그 자동화 플랜

## 블로그 정보
- URL: https://blog.naver.com/0xhenry
- 블로그명: AI 데일리 — 매일 쏟아지는 AI 뉴스 정리
- 주제: IT·컴퓨터
- 목표: 애드포스트 수익화 (50편 작성 후 신청)

## 카테고리 구조
| 카테고리 | 설명 | 발행 빈도 |
|---|---|---|
| AI 뉴스 | 당일/전일 AI 소식 속보 요약 | 매일 1-2편 |
| AI 도구 리뷰 | 새 서비스/앱 사용기 | 주 2-3편 |
| AI 활용법 | 실전 활용 가이드 | 주 1-2편 |
| AI 용어 사전 | 비개발자용 쉬운 해설 | 주 1편 |
| 주간 AI 브리핑 | 한 주 AI 뉴스 종합 | 주 1편 |

## 자동화 파이프라인

### Phase 1: 콘텐츠 생성
1. AI 뉴스 소스 수집 (RSS/웹 크롤링)
   - TechCrunch AI, The Verge AI, Ars Technica
   - OpenAI Blog, Anthropic Blog, Google AI Blog
   - X(Twitter) AI 인플루언서
   - 한국: AI타임스, 디지털데일리 AI 섹션
2. Claude가 뉴스 요약 + 블로그 포맷으로 변환
3. workspaces/naver-blog/generated/ 에 HTML 파일로 저장

### Phase 2: 네이버 발행 (claude-in-chrome 자동화)
1. blog.naver.com 로그인 상태 확인
2. 글쓰기 페이지 열기
3. 제목, 본문, 카테고리, 태그 입력
4. 발행 버튼 클릭

### Phase 3: 모니터링
- 발행된 글 목록 tracking
- 조회수/댓글 확인 (선택)

## 글 작성 가이드라인

### 네이버 SEO 최적화
- 제목: 검색 키워드를 앞에 배치 (예: "ChatGPT 5.0 출시 — 달라진 점 5가지")
- 본문: 1500-2500자 (너무 짧으면 검색 불리)
- 이미지: 최소 3장 (네이버 알고리즘이 이미지 있는 글 우대)
- 태그: 5-10개, 검색량 높은 키워드
- 소제목(H2/H3) 활용: 네이버 스마트블록 노출에 유리

### 글 구조 템플릿
```
[제목] 검색 키워드 — 핵심 내용 요약

[썸네일 이미지]

[도입] 1-2문장으로 "이게 왜 중요한지" 설명

[본문]
## 핵심 내용 1
- 설명

## 핵심 내용 2  
- 설명

## 핵심 내용 3
- 설명

[마무리] 한줄 의견 + "어떻게 생각하시나요?" (댓글 유도)

[태그] #AI #ChatGPT #인공지능 ...
```

### AI 티 안 나게 쓰는 법
- "~했습니다" 반복 금지, 문체 섞기
- 개인 의견/경험 1-2줄 삽입
- "결론적으로", "종합하면" 같은 AI 투 문구 금지
- 구어체 자연스럽게 ("솔직히 이건 좀 놀라웠어요")

## 초기 로드맵

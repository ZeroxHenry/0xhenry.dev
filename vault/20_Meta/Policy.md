# P-Reinforce 분류 정책

> 사용자 피드백이 반영되는 분류 가중치. AI가 자동으로 갱신.

## 분류 규칙

### 폴더 배치 기준
- 유사도 85% 이상 → 기존 폴더 배치
- 새로운 개념 → 상위 개념 도출 후 새 폴더 생성
- 폴더 내 파일 12개 초과 → 하위 카테고리(Recursive Sub-folder)로 세분화 제안
- 계층 깊이 3단계 이상 시 → 상위 도메인에 MOC(Map of Content) 인덱스 노트 강제 생성
- 전역 태그 시스템 적용 → 폴더 간 수평적 연결성 확보 (#topic)

### 현재 가중치

w2 (Graph Connectivity):      0.3
w3 (User Satisfaction):        0.3
